## Run
1. Crie um ambiente virtual e instale as dependências
   ~~~
   sudo apt install python3-venv
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ~~~
2. Execute o main
   ~~~
   python src/main.py
   ~~~
